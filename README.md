# Layered Exfiltration

A forensics CTF challenge testing metadata analysis and encoded-credential recovery.

## Challenge Story / Scenario

During a routine security audit, an employee's workstation was flagged for unusual activity shortly before their abrupt resignation. The machine was imaged, and investigators recovered a single photo and an encrypted archive from the desktop. The photo looks ordinary — but appearances can be deceiving. Can you recover what the employee was trying to protect?
